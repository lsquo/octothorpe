== Description ==

Octothorpe is a modern blogging theme which features a mobile first responsive layout, semantic markup and easily readable typefaces.

== Installation ==

1. Go to Appearance > Themes and click the Add New button
2. Type Octothorpe in the search form and press Enter

You can then go ahead and activate the theme.
